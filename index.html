Get Started Quickly

Algomash uses Jekyll — a free blog-aware CMS in Ruby. It can be freely hosted on GitHub Pages. No server maintenance.

Responsive Layouts

The core Algomash uses Pure.css which is a set of small, responsive CSS modules. It's only 18 KB.

Easy to Configure

Algomash is very easy to configure. Open a text editor, add your site URL and title. Start adding content.

Customize Algomash

As stated already, Algomash is a boilerplate. It's no CMS, framework or plugin. You can use this theme or customize anything if you know CSS.

Algomash is Free.

Yes, free as in freedom. You can download or fork Algomash from GitHub and get started however you wish. I've provided a neat documentation and step by step tutorial on how to create a documentation website for free. Algomash is available under GNU General Public License v2.0.
